========================================================================
    活动模板库 : MZConsole 项目概述
========================================================================

应用程序向导已为您创建了此 MZConsole 项目，作为编写可执行程序 (EXE) 的起点。

本文件概要介绍组成项目的每个文件的内容。

MZConsole.vcproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，
    其中包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

MZConsole.idl
    此文件包含项目中定义的类型库、接口和 co-class 的 IDL 定义。
    此文件将由 MIDL 编译器进行处理以生成：
        C++ 接口定义和 GUID 声明 (MZConsole.h)
        GUID 定义                                (MZConsole_i.c)
        类型库                                  (MZConsole.tlb)
        封送处理代码                                 （MZConsole_p.c 和 dlldata.c）

MZConsole.h
    此文件包含 MZConsole.idl 中定义的项目的 C++ 接口定义和 GUID 声明。它将在编译过程中由 MIDL 重新生成。

MZConsole.cpp
    此文件包含对象映射和 WinMain 的实现。

MZConsole.rc
    这是程序使用的所有 Microsoft Windows 资源的列表。


/////////////////////////////////////////////////////////////////////////////
其他标准文件：

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 MZConsole.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。

Resource.h
    这是用于定义资源 ID 的标准头文件。
